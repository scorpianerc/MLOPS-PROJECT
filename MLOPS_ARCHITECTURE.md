## ğŸ“Š Metrics & Monitoring

### Model Metrics (Tracked by MLflow)
- **Training Metrics**: Loss, Accuracy per epoch
- **Validation Metrics**: Accuracy, Precision, Recall, F1-score
- **Test Metrics**: Final accuracy, Confusion matrix
- **Model Artifacts**: Weights, Config, Plots

### API Metrics (Prometheus)
- **predictions_total**: Counter for total predictions
- **prediction_latency_seconds**: Histogram of response times
- **prediction_confidence**: Gauge of confidence scores
- **api_errors_total**: Counter for errors
- **api_requests_total**: Counter for all requests

### Drift Metrics
- **Data Drift**: KS statistic, p-value for numerical features
- **Categorical Drift**: Chi-square statistic, p-value
- **Performance Drift**: Accuracy change from baseline
- **Trend**: Linear regression slope over 30 days
